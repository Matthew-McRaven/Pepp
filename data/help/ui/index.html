User Interface Overview

This section describes the operation of each UI component in the Pepp. Each component is documented separately in its own page. Each page focuses on what the component does and how to interact with it. The workflows involving these components are covered in later sections.

These symbols are used to indicate the availability of each component:

Symbol State Description
● Visible by default The component is present and visible by default.
○ Hidden by default The component exists but is not visible by default; it can be enabled via the pane visibility controls.
Not available The component is not visible by default nor can it be enabled via the pane visibility controls.

Object Code Editor

The object code editor is a text editor for entering program bytes directly in hexadecimal format. It accepts only valid hexadecimal characters and the sentinel characters z and Z.

In a feature new to Pep/10, both the sentiel and whitespace are optional. Both are automatically inserted when needed by the Pep/10 operating system. This improvement has been transparently backported to Pep/8 and Pep/9.

It features an overwrite mode, which can be toggled via the Insert key. This allows you to replace existing bytes without shifting the surrounding content.

As an editor, it is always read-only in Debugger mode. Outside of ISA3, it is also read-only because it is a generated output from assembling Pep/10 source programs.

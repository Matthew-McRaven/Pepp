Symbol Table

The symbol table lists all symbols defined and used in your program, along with their associated values. It primarily appears in the Debugger mode of assembly projects, but may be made visible in the Editor using the visibility controls.

Note: In OS4 projects, the table automatically switches between OS and user symbols depending on the current value of the program counter (PC).

Selection

You can click to select a single symbol, then shift-click to select a rectangular block. Your platform’s “select all” shortcut also works. If you use File → Copy or your platform’s copy shortcut, a text representation of the symbol table is copied to your clipboard.

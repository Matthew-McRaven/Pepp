Automatic Formatting

Automatic formatting rewrites your program using a standard style. This feature is available at all levels of abstraction; the exact menu label may vary by level.

Applies to: Mc2, ISA3, Asmb3, OS4, Asmb5

Menu Names by Level

The formatting command appears under the Edit menu. The label can differ by level:

Level of Abstraction Menu Path
Mc2 (Microcode) Edit → Format Microcode
ISA3 (Machine Language) Edit → Format Object Code
Asmb3, OS4, Asmb5 Edit → Format Source Code

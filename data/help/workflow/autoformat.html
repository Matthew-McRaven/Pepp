Automatic Formatting

Applies to: Mc2, ISA3, Asmb3, OS4, Asmb5

Automatic formatting rewrites your program using a standard style. Formatting affects indentation, capitalization of mnemonics, and syntax styling. Automatic formatting does not change the written code (e.g., change LDWA to STWA). Any errors in syntax will not be affected by formatting. However, automatic formatting makes it is easier to review written code, which helps speed up the debugging process.

This feature is available at all levels of abstraction; the exact menu label may vary by level.

Menu Names by Level

The formatting command appears under the Edit menu. The label can differ by level:

Level of Abstraction Menu Path
Mc2 (Microcode) Edit → Format Microcode
ISA3 (Machine Language) Edit → Format Object Code
Asmb3, OS4, Asmb5 Edit → Format Source Code
